Panduan Setup Google Drive

Agar foto di dalam folder Google Drive dapat dipilih oleh pengguna, Anda perlu membuat proxy Apps Script gratis di akun Google Anda:

  1. Buka script.google.com dan login dengan akun Google Anda.
  2. Klik tombol "Project Baru".
  3. Hapus kode bawaan, lalu salin dan tempel kode berikut: 
    function doGet(e) {
  var folderId = e.parameter.id;
  if (!folderId) {
    return ContentService.createTextOutput(JSON.stringify({success: false, error: "Missing id"})).setMimeType(ContentService.MimeType.JSON);
  }
  try {
    var folder = DriveApp.getFolderById(folderId);
    var files = folder.getFiles();
    var subfolders = folder.getFolders();
    var resultFiles = [];
    var resultFolders = [];
    while (subfolders.hasNext()) {
      var sub = subfolders.next();
      resultFolders.push({ id: sub.getId(), name: sub.getName() });
    }
    while (files.hasNext()) {
      var file = files.next();
      var mime = file.getMimeType();
      if (mime.indexOf("image/") !== -1) {
        resultFiles.push({ id: file.getId(), name: file.getName(), url: "https://lh3.googleusercontent.com/d/" + file.getId() });
      }
    }
    return ContentService.createTextOutput(JSON.stringify({success: true, files: resultFiles, folders: resultFolders, folderName: folder.getName()})).setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({success: false, error: err.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}
  4. Klik Terapkan (Deploy) > Penerapan baru (New deployment).
  5. Klik ikon gerigi > pilih Aplikasi web (Web app).
  6. Setel Jalankan sebagai (Execute as) ke: "Saya" (Me).
  7. Setel Siapa yang memiliki akses (Who has access) ke: "Siapa saja" (Anyone).
  8. Klik Terapkan (Deploy) dan setujui izin akses akun Google Anda jika diminta.
  9. Salin URL Aplikasi Web (Web App URL) yang diberikan, lalu paste ke kolom URL Script di form Admin.
  10. Pastikan folder Google Drive Anda disetel share-nya ke: "Siapa saja yang memiliki link dapat melihat" (Anyone with link can view).
